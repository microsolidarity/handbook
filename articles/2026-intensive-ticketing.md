# Summer Intensive 2026 Ticketing Page

The ticket price range depends on the type of room you select and on the amount you chose to contribute towards the training itself.&#x20;

Select **Abundant** if money is not an issue for you and select a **Supported** ticket if money is tighter these days. Alternatively, select **Standard** for the in-between option.

**Note**: when you select a ticket, Open Collective automatically will automatically add a 15% tip onto the transaction. This is _optional_ and can be disabled.
